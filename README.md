# SpringClient

This is a simple REST CRUD API built using Java, Spring, JPA and MySQL as its databases. It allows users to perform the basic CRUD operations on a 'item' resource. The API supports the following endpoints:

POST /items: Creates a new item
GET /items: Retrieves a list of all items
GET /items/:id: Retrieves a specific item by its id
PUT /items/:id: Updates an existing item by its id
DELETE /items/:id: Deletes an existing item by its id
